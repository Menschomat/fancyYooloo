Teamverzeichnis

Auf alle Dateien in diesem Verzeichnis können Sie über den Webbrowser zugreifen.

Um Daten in diesem Verzeichnis abzulegen, rufen Sie im Browser die Datei upload.php in diesem Verzeichnis auf.
Sie erhalten damit ein Formular mit dem Sie die wichtigsten Dateioperationen (Hochladen, Löschen, Verzeichnis erstellen etc.) durchführen können.
Hierbei ist jedoch zu beachten, dass in den meisten Umgebungen die Größe einer Datei für das Hochladen auf begrenzt ist.
Dieser Wert kann nur durch den Administrator geändert werden.

 

Hinweise

Alle Dateien, die vom System angelegt wurden, können nicht verändert oder gelöscht werden.

Beachten Sie, dass jeder, der Zugriff auf dieses Verzeichnis hat, jede Datei verändern und löschen kann.

Der Administrator des Teams kann jedoch festlegen (s. Löschen und Überschreiben nur dem Admin erlauben),
dass vorhandene Dateien nicht von anderen gelöscht bzw. verändert werden können.
